CSS（级联样式表）：整个结构称为规则集

1、水平居中图像（这个我总是忘记）
img {
display: block;
margin: 0 auto;
}
2、CSS中各个属性是处在不同的模块中，只有特定的标签（元素）才能够引用这些模块
3、标签中rel定义当前文档与被链接文档之间的关系
4、CSS 语言有规则来控制在发生冲突时哪个选择器更强。这些规则被称为级联和特异性，特异性经常导致后来写的样式会无法覆盖前面写的样式导致不可预测的错误，因此要注意规范
5、函数：CSS中还有一些简单的函数可供数学运算
6、CSS @rules（发音为"在规则中"）提供了 CSS 应执行什么或应如何行为的指令。有些@rules一个关键字和一个值就很简单。例如，将样式表导入另一个 CSS 样式表：@import
7、速记：某些属性，如字体、背景、填充、边框和边距，称为速记属性。这是因为速记属性在一行中设置了多个值。
8、CSS实际工作原理，详情见https://developer.mozilla.org/en-US/docs/Learn/CSS/First_steps/How_CSS_works
