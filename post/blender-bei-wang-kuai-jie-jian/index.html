Blender备忘快捷键

2021-02-05

Shift+A是添加，Shift+D是复制，G是移动，R是旋转，Delete是删除
F3查询栏 空格渲染动画
S = scale G = translate R = rotation
以上三个可以配合X,Y,Z使用，按shift+z 是可以去除z轴的变化
table进入编辑模式
编辑模式shift复选 alt选中一圈边
o进入等比例编辑模式，同时鼠标中建可以改变缩放范围
像这类模式屏幕中间上面选

还有C，B（中建取消选择）多选
同时双面选 在右上角，Z是快捷键
平视和透视的变换？
shift+z切换渲染视图（网格）
z可以选择视图的各种模式
f12打开后的渲染窗口
H隐藏，alt+h显示隐藏
shift+f3 节点编辑模式 shift+f5 返回3D视图这些模式改变在左上角可以改
ctrl+r加紫线，建模用
m移动一个层的物体到另一个层
选中一圈点 i插入一个面，做马克杯顶可用
选择四个点or？ 按f增加面 删除面的时候要选face
e拉伸模式
重新计算recalculate shift+N
物品scale完的时候ctrl+a apply scale 选择应用比例，不然粒子等效果会被改变
对物体右键 shading smooth
jonathan williamson 擅长建模
blender cookie or cg cookie blender

shift+s cursor to select 让3d光标吸附物体
选中物体的边 按ctrl+E 选择mark seam 标记缝，uv展开用
选中物体 打开uvimage edit u-》unwrap 让材质上去
ctrl+shift+n 删除法线 ，在加了固化modifier里面选only rim就可以得到正确的方向
或者老方法 法线唤出在右上角show overlay那里
教程里左边工具栏就是2.8左上角view select add mesh 那些
mesh-》normal-》flip翻转面的方向 或者alt+n-》flip 还要在固化modif改offset为-1
选中物体 ctrl+L 可以复制相关信息 包括材质
刷权重模式 ctrl+tab
ctrl+j 合并物体，但要apply 相应的modifier来让它应用到合并起来的物体上

锥形灯光 use？cone可以让光与其他物体发生交互，光的size用来调节阴影的柔和程度，越高越柔和

object面板中可以将物体从组里去掉
focal length 是焦距，就是摄像机放大差不多。。
focus可以选择对焦的物体
点开limits，改动distance就是改变焦点
